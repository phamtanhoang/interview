## 🧭 Cách ôn hiệu quả

- **Tự trả lời trước**, rồi mới mở khối đáp án để đối chiếu — phỏng vấn kiểm tra khả năng *giải thích*, không phải đọc thuộc.
- Diễn đạt lại bằng lời của bạn; gắn với **ví dụ từ dự án thật** của mình.
- Câu hỏi **hành vi (behavioral)** trả lời theo cấu trúc **STAR** (Situation → Task → Action → Result).
- Mỗi file đều có **mục lục bấm-nhảy** ở đầu và **checklist ôn nhanh** ở cuối (trừ backend).

## 📐 Định dạng

- Tiếng Việt có dấu; giữ nguyên tiếng Anh cho thuật ngữ/tên công nghệ/code.
- Đáp án dạng **gạch đầu dòng** súc tích + tối đa **1 ví dụ ngắn** (code/bảng).
- Hiển thị đẹp nhất trên **GitHub** hoặc **VS Code preview** (`Ctrl/Cmd+Shift+V`).
